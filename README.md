### **💡 프로젝트 소개**

<br/>

- vanilla JavaScript 를 이용한 당근 찾기 게임

1. 게임 시작 버튼 클릭
2. 시간 내에 당근을 클릭하면 WIN !

<br/>

---

<br/>

### **🔧 기능**

- play 버튼 클릭 시

  - 타이머 시작
  - item 필드에 랜덤 배치
  - 클릭 할 당근 수 표시
  - 배경음 시작
  - 게임 중단 버튼 생성

    <br/>

- 게임 중단 버튼 클릭 시,

  - replay modal 창 생성
  - 타이머 초기화

    <br/>

- 게임 패배 ( 시간 초과⏰ OR 벌레🐛 클릭 )

  - 패배 modal창 생성
  - replay 버튼 생성

    <br/>

- 게임 승리 ( 시간 내 당근🥕 모두 클릭 )

  - 승리 modal창 생성
  - replay 버튼 생성
